## ⚡️ Installation

Clone this repo and install the dependencies using Yarn.

```bash
yarn install
```

For iOS, install the pods:

```bash
cd ios && pod install && cd ..
```

## 📲 Local Usage
For iOS, run:

```bash
yarn ios
```

For Android, run:

```bash
yarn android-
```
